## ğŸ› Troubleshooting

### Common Issues

**1. Canvas module not found**
Solution: Make sure Node.js 18.x is set in Vercel

**2. API returns 401**
Solution: Check X-API-Key header is being sent

**3. Stripe webhook fails**
Solution: Verify webhook secret in environment variables

**4. Images look bad**
Solution: Adjust font sizes in server.js
