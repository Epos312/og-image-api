## ⚡ ЗАПУСК ЗА 3 ШАГА (2 ЧАСА)

### ШАГ 1: Подготовка (15 мин)
```
□ Создать GitHub аккаунт → github.com
□ Создать Vercel аккаунт → vercel.com (войти через GitHub)
□ Создать Stripe аккаунт → stripe.com
```

### ШАГ 2: Деплой (1 час)
```
□ Распаковать архив og-image-saas-final.tar.gz
□ Создать репозиторий на GitHub
□ Загрузить все файлы в репозиторий
□ В Vercel нажать "Import Project"
□ Выбрать ваш репозиторий
□ Нажать "Deploy" (займёт 2 минуты)
```

### ШАГ 3: Stripe настройка (45 мин)
```
□ В Stripe создать 3 продукта ($9, $29, $99)
□ Скопировать Price IDs
□ Создать Webhook
□ В Vercel добавить переменные окружения
□ Redeploy проект
```

**ГОТОВО! Сайт работает.**
